## Overview

This repository contains a fully reproducible computational pipeline for the analysis of short tandem repeat (STR) motifs in the human genome, as described in the paper:

**"Distinct evolutionary signatures shape depletion and conservation of short tandem repeat motifs in the human genome"**

The pipeline performs genome-wide STR profiling, comparative primate conservation analysis, chromosome X repeat architecture analysis, and cell-type-specific expression mapping.

## Methods Overview

### Phase 1: STR Identification

STRs were identified using Tandem Repeats Finder (TRF v4.09) with parameters:

- match = 2
- mismatch = 7
- delta = 7
- PM = 80
- PI = 10
- minscore = 50
- maxperiod = 2000

Custom Python scripts parsed TRF outputs and extracted repeat coordinates, length, unit size, alignment scores, and sequence composition.

### Phase 2: Chromosome X Compartment Analysis

STR loci were assigned to inactive (Xi) and active (Xa) compartments using published X-inactivation domain annotations. Genomic overlap analyses were performed using **bedtools intersect**.

### Phase 3: Chromosome X vs Autosome Comparisons

Chromosome X was compared with chromosomes 1, 8, 19, and 21 using five STR-associated features:

- Repeat length
- Repeat period size
- TRF alignment score
- Repeat copy number
- Sequence entropy
### Phase 4: Evolutionary Conservation Analysis

All possible 4-bp and 5-bp motifs (1,280 total) were analyzed using a third-order Markov model.

The analysis included:

- Estimation of expected motif frequencies using a third-order Markov model
- Calculation of observed-to-expected (O/E) ratios
- Genome-wide depletion analysis
- PhyloP conservation scoring across five primate species
- Identification of evolutionarily conserved non-CG motifs

The primate genomes included:

- Human
- Chimpanzee
- Gorilla
- Orangutan
- Macaque

---

### Phase 5: Functional Association With Gene Expression

A comparative case-control framework evaluated whether genes located near evolutionarily conserved non-CG motifs exhibit distinct expression patterns.

Expression data were obtained from the CZ CELLxGENE (March 2025 release) across six representative cell types:

- Neurons
- Keratinocytes
- Pancreatic beta cells
- Hepatocytes
- T cells
- Fetal cardiomyocytes

## Usage

### Step 1 — Prepare Input Data

Place the required TRF output files into:

```text
data/intermediate/
```

Required files:

- chr1.dat
- chr8.dat
- chr19.dat
- chr21.dat
- chrX.dat

If these files are unavailable, generate them using TRF from the corresponding chromosome FASTA sequences (not distributed with this repository because of file size).

---

### Step 2 — Run the Complete Pipeline

```bash
cd D:\STR_Final_Test\github_repo

$env:PYTHONUTF8="1"

python run_pipeline.py
```

All reproducible outputs are generated automatically inside

- Figures (PNG, TIFF, PDF)
- Tables (TSV, CSV)
- Summary statistics
- Evolutionary conservation reports

## Validation

The computational pipeline has been validated through reverse engineering and reproducibility testing.

Major validation steps include:

- Verification of the complete repository structure.
- Validation of all primary analysis scripts.
- Reconstruction of supplementary tables.
- Validation of observed-to-expected (O/E) ratio calculations.
- Reproducibility testing of figure generation.
- Cross-validation of generated outputs against manuscript results.

The file

`final_output/tables/all_kmers_oe_ratios.tsv`

was validated against the original computational analysis and showed an **exact match** with the reported results.
